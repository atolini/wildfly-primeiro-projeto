Getting Started
---------------
O projeto precisa do JDK 19, uma instância do MySQL na porta padrão com a base 'vendasdb'. 

O projeto já tem configurado um DataSource. 

Caso queira acessar o painel administrador do WildFly realize o cadastro de um usuario. 

Starting a Standalone Server
----------------------------
A WildFly standalone server runs a single instance.

<JBOSS_HOME>/bin/standalone.sh      (Unix / Linux)

<JBOSS_HOME>\bin\standalone.bat     (Windows)
